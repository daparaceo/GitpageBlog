## 디딤돌대출이란?

디딤돌대출은 **국토교통부 주택도시기금**이 운영하는 저금리 주택 구입 자금 대출입니다. 무주택 서민과 실수요자를 위해 시중 금리보다 낮은 이자율을 적용하며, 생애최초 구입자에게는 추가 우대금리까지 제공합니다.

2024년부터 신생아 특례 디딤돌대출이 신설되어 자녀를 출산한 가구는 더 낮은 금리로 이용할 수 있게 되었습니다. 주거 안정을 원하는 실수요자라면 가장 먼저 살펴봐야 할 대출 상품입니다.

## 신생아 특례 디딤돌대출 별도 안내

2024년 1월 출시된 **신생아 특례 디딤돌대출**은 일반 디딤돌대출과 별도로 운영됩니다. 출산 가구를 위한 파격적인 조건이 적용되어 관심이 높습니다.

| 항목 | 조건 |
|------|------|
| 신청 자격 | 대출 신청일 기준 2년 이내 출산한 무주택 가구 |
| 소득 기준 | 부부합산 연 **1억 3,000만 원 이하** |
| 자산 기준 | 순자산 **4억 6,900만 원 이하** |
| 주택 가격 | **9억 원 이하** |
| 대출 한도 | 최대 **5억 원** |
| 금리 | **연 1.6~3.3%** (소득 구간별 차등) |
| 추가 자녀 출산 시 | 자녀 1인당 금리 **-0.2%p** 추가 인하 (최저 1.2%) |

신생아 특례 대출은 입양 자녀도 인정되며, 쌍둥이를 포함한 다태아의 경우 자녀 수에 맞게 우대 혜택이 적용됩니다.

## 대출 한도 계산 방법

대출 한도는 단순히 최대 금액인 **4억 원(신생아 특례 5억 원)**이 자동 적용되는 것이 아닙니다. 실제 한도는 다음 세 가지 중 가장 낮은 금액으로 결정됩니다.

1. **LTV(담보인정비율)** — 주택 가격의 최대 **70%** (생애최초는 **80%**)
2. **DTI(총부채상환비율)** — 연간 원리금 상환액이 연소득의 **40~60%** 이내
3. **상품별 한도 상한** — 일반 **4억 원**, 신생아 특례 **5억 원**

예를 들어 매매가 **4억 원** 주택을 생애최초로 구입하는 경우:
- LTV 80% 적용 시 → 최대 **3억 2,000만 원**
- 소득이 충분하다면 한도 내에서 원하는 금액 선택 가능

## 신청 방법 단계별 안내

### 1단계 — 대출 자격 사전 확인

[주택도시기금 포털](https://nhuf.molit.go.kr)에서 소득·자산 기준을 먼저 확인합니다. 공인인증서 또는 간편인증으로 로그인 후 대출 자격 조회 메뉴를 이용하면 됩니다.

### 2단계 — 매매계약 체결

자격 확인 후 원하는 주택의 매매계약을 체결합니다. 계약서에는 잔금일과 등기 예정일이 명시되어야 합니다.

### 3단계 — 대출 신청

다음 취급 기관 중 한 곳을 방문하거나 온라인으로 신청합니다.

| 취급 기관 | 신청 방법 |
|-----------|---------|
| 우리은행 | 영업점 방문 또는 우리WON뱅킹 앱 |
| KB국민은행 | 영업점 방문 또는 KB스타뱅킹 앱 |
| 신한은행 | 영업점 방문 또는 신한SOL 앱 |
| NH농협은행 | 영업점 방문 또는 NH올원뱅크 앱 |
| IBK기업은행 | 영업점 방문 또는 IBK 앱 |

### 4단계 — 서류 제출

| 서류 종류 | 세부 내용 |
|-----------|---------|
| 신분증 | 주민등록증, 운전면허증 등 |
| 주민등록등본 | 발급일 **3개월 이내** |
| 가족관계증명서 | 발급일 **3개월 이내** |
| 소득 증빙 | 근로소득원천징수영수증, 사업소득 확인서 등 |
| 매매계약서 | 공인중개사 날인 원본 |
| 건물등기부등본 | 발급일 **1개월 이내** |
| 재직증명서 | 직장인의 경우 |

### 5단계 — 심사 및 대출 실행

서류 제출 후 통상 **5~10 영업일** 이내에 심사 결과가 통보되며, 잔금 납부일에 맞춰 대출금이 실행됩니다.

## 중도상환수수료·상환 방식

| 항목 | 내용 |
|------|------|
| 상환 방식 | 원리금균등상환, 원금균등상환, 체증식상환 중 선택 |
| 중도상환수수료 | 대출 실행 후 **3년 이내** 상환 시 **1.2%** |
| 3년 이후 중도상환 | **수수료 없음** |
| 대출 기간 | 10년·15년·20년·30년 선택 |

체증식상환은 초반에 원금 상환 비율이 낮고 나중에 높아지는 방식으로, 초기 상환 부담이 적어 신혼부부나 사회초년생에게 유리합니다.

## 자주 묻는 질문 (FAQ)

**Q. 전세에서 매매로 전환할 때도 신청 가능한가요?**
네, 가능합니다. 단, 무주택 요건을 충족해야 하고, 기존 전세대출을 디딤돌대출로 대환하는 것은 불가합니다.

**Q. 분양권 상태에서도 신청할 수 있나요?**
분양권 취득 후 잔금 납부 시점에 신청할 수 있습니다. 단, 분양가와 현재 감정가 중 낮
