매년 5월이 되면 종합소득세 신고 때문에 머리가 아프다는 분들이 많습니다. 그런데 사실 국세청에서 이미 세금 계산을 다 해 두고 확인만 기다리는 납세자가 수백만 명에 달합니다. 바로 **모두채움 신고** 대상자입니다. 해당된다면 홈택스에 접속해 내용 확인 후 클릭 몇 번으로 신고가 끝납니다.

> **안내:** 이 글은 2026년 4월 25일 기준으로 작성되었습니다. 세법 개정 및 국세청 시스템 변경에 따라 세부 내용이 달라질 수 있으므로, 신고 전 반드시 [국세청 홈택스](https://www.hometax.go.kr) 및 [국세청 공식 안내](https://www.nts.go.kr)를 확인하시기 바랍니다.

## 모두채움 신고란?

모두채움 신고는 국세청이 납세자의 수입·지출 자료를 미리 수집해 세액까지 계산한 뒤, 납세자가 내용을 확인하고 제출만 하면 신고가 완료되는 서비스입니다. 복잡한 입력 과정 없이 **내용 확인 → 동의 → 제출** 3단계로 끝납니다.

2025년 귀속 기준으로 약 **700만 명**이 모두채움 대상자로 분류되어 안내문을 받게 됩니다.

### 일반 신고와 모두채움 신고 비교

| 구분 | 일반 신고 | 모두채움 신고 |
|------|-----------|--------------|
| 수입 입력 | 직접 입력 | 국세청 자동 채움 |
| 공제 항목 | 직접 선택·입력 | 기본공제 자동 반영 |
| 소요 시간 | 30분~수 시간 | **5~15분** |
| 대상 | 모든 종합소득세 신고자 | 단순경비율 대상자 등 요건 충족자 |
| 세무 지식 필요도 | 높음 | 낮음 |

## 모두채움 신고 대상자 확인

모두채움 서비스 대상이 되는지 먼저 확인해야 합니다.

### 주요 대상

| 유형 | 내용 |
|------|------|
| 단순경비율 적용 사업소득자 | 업종별 수입금액 기준 이하인 소규모 사업자 |
| 기타소득만 있는 경우 | 강연료·원고료 등 기타소득 수입금액 **300만 원 이하** |
| 근로소득 + 소규모 사업소득 | 일부 복합소득자 |
| 금융소득 분리과세 대상 | 이자·배당소득 합계 **2,000만 원 이하** |

### 대상 제외

- 장부 작성 대상 사업자 (복식부기·간편장부 의무자)
- 부동산 임대소득이 있는 경우 (별도 신고)
- 성실신고확인 대상 사업자
- 해외 소득이 있는 경우

> **확인 방법:** 홈택스 로그인 후 메인 화면 상단 배너 **"모두채움 안내문"** 확인, 또는 **My홈택스 → 세금신고 → 종합소득세 → 모두채움 신고** 진입 시 대상 여부 자동 안내

## 홈택스 모두채움 신고 단계별 방법

### 1단계 — 홈택스 접속 및 로그인

[홈택스(www.hometax.go.kr)](https://www.hometax.go.kr)에 접속 후 로그인합니다.

- 공동인증서(구 공인인증서)
- 금융인증서
- 간편인증 (카카오, 네이버, 패스 등)
- 아이디·비밀번호

간편인증으로도 신고가 가능하므로 공동인증서가 없어도 됩니다.

### 2단계 — 모두채움 신고 진입

홈택스 메인 화면 상단의 **"세금신고"** 메뉴 클릭 → **"종합소득세 신고"** → **"모두채움 신고/단순경비율 신고"** 선택

또는 메인 배너의 **"모두채움 안내문"** 바로가기 클릭

### 3단계 — 미리채워진 내용 확인

국세청이 자동으로 수집한 항목을 화면에서 확인합니다.

| 확인 항목 | 내용 |
|-----------|------|
| 수입금액 | 사업소득, 기타소득, 근로소득 등 |
| 경비 | 단순경비율 자동 계산액 |
| 기본공제 | 본인 공제, 부양가족 공제 |
| 세액 | 산출세액, 기납부세액, 납부(환급)세액 |

수입금액이 실제와 다르거나 누락된 항목이 있다면 **직접 수정**할 수 있습니다. 이 경우 일반 신고 방식으로 전환됩니다.

### 4단계 — 공제 항목 추가 입력 (선택)

모두채움 화면에서 아래 항목은 추가로 입력하면 절세 효과가 있습니다.

| 추가 가능 공제 | 비고 |
|---------------|------|
| 노란우산공제 납입액 | 소기업·소상공인 해당 |
| 연금보험료 공제 | 국민연금, 지역가입자 |
| 의료비·교육비 | 일부 소득 유형에서 가능 |
| 기부금 | 종교단체, 지정기부금 등 |

### 5단계 — 신고서 제출

내용 확인 완료 후 하단의 **"신고서 제출하기"** 버튼 클릭 → 제출 완료 화면에서 **접수증** 확인 및 저장

접수번호가 부여되면 신고가 정상 완료된 것입니다.

### 6단계 — 납부 또는 환급 확인

| 구분 | 방법 |
|------|------|
| 세금 납부 | 홈택스 내 **"납부하기"** → 계좌이체·카드 납부 가능 |
| 분납 신청 | 납부세액 **1,000만 원 초과** 시 2개월 분납 가능 |
| 환급 | 신고 시 등록한 환급계좌로 **30일 이내** 입금 |

> **분납 기한:** 1차 5월 31일, 2차 **7월 31일**

## 손택스(모바일)로 신고하는 방법

PC가 불편하다면 스마트폰 앱 **손택스**를 활용할 수 있습니다.

1. 앱스토어 또는 구글플레이에서 **"손택스"** 검색 후 설치
2. 간편인증(카카오·네이버·패스) 로그인
3. 홈 화면 → **"종합소득세 신고"** → **"모두채움 신고"**
4. 이후 과정은 PC 홈택스와 동일

손택스는 화면이 단순화되어 있어 처음 신고하는 분들도 비교적 쉽게 이용할 수 있습니다.

## ARS 전화 신고 — 가장 간단한 방법

모두채움 안내문을 받은 경우, ARS 전화 신고도 가능합니다.

| 항목 | 내용 |
|------|------|
| 전화번호 | **국번 없이 126** → 1번(세금신고·납부) → 1번(종합소득세) |
| 이용 시간 | 평일 오전 9시 ~ 오후 6시 |
| 필요 정보 | 주민등록번호, 사업자등록번호(있는 경우) |
| 가능 대상 | 모두채움 안내문 수령자 중 단순 확인 신고자 |

ARS는 미리채워진 내용에 동의하는 경우에만 이용 가능합니다. 수정이 필요하면 홈택스 또는 손택스를 이용해야 합니다.

## 신고 후 꼭 확인할 것들

### 신고 결과 조회

홈택스 → **My홈택스** → **"신고납부 내역"** → 종합소득세 신고 이력 확인

### 환급계좌 사전 등록

환급금이 발생하는 경우, 환급계좌가 등록되지 않으면 지급이 지연됩니다.

홈택스 → **My홈택스** → **"환급계좌 개설·변경"** 에서 미리 등록해 두세요.

### 가산세 주의사항

| 위반 유형 | 가산세율 |
|-----------|---------|
| 무신고 (일반) | 납부세액의 **20%** |
| 무신고 (부정행위) | 납부세액의 **40%** |
| 납부 지연 | 1일당 **0.022%** (연 약 8%) |
| 과소신고 | 과소신고세액의 **10%** |

## 자주 묻는 질문 (FAQ)

**Q. 안내문을 받지 못했는데 모두채움 신고를 할 수 있나요?**
홈택스에 로그인 후 종합소득세 신고 메뉴에서 직접 확인하면 됩니다. 안내문 발송 여부와 무관하게 대상자라면 모두채움 화면이 제공됩니다.

**Q. 모두채움 내용이 실제와 다를 때는 어떻게 하나요?**
화면에서 직접 수정할 수 있습니다. 수정 후 제출하면 일반 신고로 처리됩니다. 수입이 누락된 경우 반드시 추가 입력해야 가산세를 피할 수 있습니다.

**Q. 이미 연말정산을 했는데 종합소득세도 신고해야 하나요?**
근로소득만 있고 회사에서 연말정산을 완료했다면 별도 신고 불필요합니다. 단, 근로소득 외 다른 소득(사업, 기타소득 등)이 있다면 반드시 종합소득세 신고를 해야 합니다.

**Q. 환급금은 언제 들어오나요?**
신고 후 통상 **30일 이내** 입금됩니다. 신고가 몰리는 5월 말에 신고하면 다소 늦어질 수 있으므로 5월 초에 신고하는 것이 유리합니다.
