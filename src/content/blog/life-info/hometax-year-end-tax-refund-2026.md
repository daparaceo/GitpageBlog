매년 2월이면 직장인들은 연말정산 결과를 기다립니다. 하지만 정작 환급금이 언제 들어오는지, 내 환급액이 맞게 계산됐는지 확인하는 방법을 모르는 분이 많습니다. 이 글에서는 홈택스에서 환급금을 조회하는 방법부터 환급이 잘못 처리됐을 때 추가 경정청구하는 방법까지 단계별로 정리했습니다.

> **안내:** 이 글의 환급 일정·금액 기준은 2025년 귀속 연말정산(2026년 신고) 기준입니다. 세법 개정 또는 국세청 일정 변경에 따라 내용이 달라질 수 있으므로, 정확한 정보는 [국세청 홈택스(hometax.go.kr)](https://www.hometax.go.kr)에서 직접 확인하시기 바랍니다.

## 연말정산 환급금이란?

연말정산은 1년간 납부한 근로소득세를 정산하는 절차입니다. 원천징수된 세금이 실제 내야 할 세금보다 많으면 **환급**, 적으면 **추가 납부**가 발생합니다.

| 구분 | 내용 |
|------|------|
| 환급 발생 이유 | 공제 항목(의료비·교육비·기부금 등) 반영 후 세금이 줄어든 경우 |
| 추가 납부 발생 이유 | 공제 항목이 적거나 부양가족 변동으로 세금이 늘어난 경우 |
| 환급금 지급 주체 | 회사(근무지) — 국세청이 아닌 회사가 직원에게 지급 |
| 일반적 환급 시기 | 매년 **2월 급여일** 또는 **3월 급여일** |

핵심은 환급금이 국세청에서 직접 계좌로 입금되는 것이 아니라, **회사가 근로자에게 급여와 함께 지급**한다는 점입니다. 회사는 국세청에 환급 신청을 하고, 국세청이 회사에 환급해 주면 회사가 직원에게 전달하는 구조입니다.

## 홈택스에서 환급금 조회하는 방법

### 1단계: 홈택스 로그인

[홈택스(hometax.go.kr)](https://www.hometax.go.kr)에 접속한 뒤 공동인증서, 금융인증서, 간편인증(카카오·네이버·PASS 등) 중 하나로 로그인합니다.

### 2단계: 지급명세서 조회

로그인 후 아래 경로로 이동합니다.

> **[My 홈택스] → [지급명세서·장려금·연금저축 등 조회] → [근로소득 지급명세서]**

여기서 귀속 연도를 **2025년**으로 선택하면, 회사가 제출한 연말정산 결과를 확인할 수 있습니다.

| 확인 항목 | 위치 |
|-----------|------|
| 결정세액 | 지급명세서 하단 |
| 기납부세액(원천징수액) | 지급명세서 하단 |
| 차감징수세액(환급 또는 추가납부액) | 결정세액 - 기납부세액 |

**차감징수세액이 음수(마이너스)이면 환급**, 양수이면 추가 납부입니다.

### 3단계: 환급금 상세 조회

더 직접적인 환급금 조회는 아래 경로에서 가능합니다.

> **[세금신고] → [근로소득 지급명세서 제출 현황]**
> 또는
> **[조회/발급] → [세금 신고납부] → [환급금 조회]**

조회 화면에서 환급 예정 금액과 처리 상태를 확인할 수 있습니다.

## 환급금 지급 일정

| 시기 | 내용 |
|------|------|
| **1~2월** | 회사가 연말정산 서류 수집 및 정산 진행 |
| **2월 말** | 대부분의 회사가 2월 급여에 환급금 포함 지급 |
| **3월** | 2월 처리 못한 회사의 추가 지급 |
| **6월 이후** | 근로소득 확정 신고 후 수정된 환급금 정산 |

회사마다 지급일이 다르므로 급여 담당자에게 문의하는 것이 가장 확실합니다. 단, **중도 퇴사자·이직자**는 환급 방식이 다를 수 있습니다.

---

## 중도 퇴사자·이직자의 환급금 처리

| 상황 | 처리 방법 |
|------|-----------|
| 연도 중 퇴사 후 미취업 | 다음 해 **5월 종합소득세 신고** 기간에 직접 신고·환급 신청 |
| 퇴사 후 같은 해 재취업 | 새 직장에서 전 직장 원천징수영수증 합산하여 연말정산 |
| 현재도 퇴사 상태 | 홈택스에서 직접 근로소득 신고(5월) 후 환급 신청 |

중도 퇴사자는 종합소득세 신고 기간인 **5월 1일~5월 31일**에 직접 홈택스에서 신고해야 환급을 받을 수 있습니다. 신고하지 않으면 환급금이 자동으로 지급되지 않습니다.

---

## 환급금이 적거나 잘못됐을 때 — 경정청구

연말정산을 마친 뒤 공제 항목을 빠뜨렸거나 잘못 계산된 경우, **경정청구**를 통해 추가 환급을 받을 수 있습니다.

### 경정청구란?

신고·납부한 세금이 실제보다 많을 때 돌려달라고 국세청에 요청하는 제도입니다. 법적으로 **신고 기한 후 5년 이내**에 신청할 수 있습니다.

### 경정청구 신청 방법

> **홈택스 → [세금신고] → [근로소득세 신고] → [경정청구]**

| 단계 | 내용 |
|------|------|
| 1 | 홈택스 로그인 후 경정청구 메뉴 진입 |
| 2 | 귀속 연도 선택 (2025년) |
| 3 | 수정할 공제 항목 입력 (의료비·교육비·기부금 등) |
| 4 | 변경된 세액 확인 후 제출 |
| 5 | 국세청 검토 후 환급 (보통 **2개월 이내**) |

### 자주 빠뜨리는 공제 항목

| 공제 항목 | 내용 |
|-----------|------|
| 의료비 | 안경·렌즈 구입비, 한의원·치과 포함 |
| 교육비 | 학원비(취학 전 아동), 대학원 등록금 |
| 기부금 | 종교단체 기부, 사회복지 기부 |
| 월세 세액공제 | 총급여 **8,000만 원 이하**, 국민주택규모 이하 |
| 신용카드 공제 | 전통시장·대중교통·도서·공연 사용분 추가 공제 |

## 홈택스 모바일(손택스)에서 조회하는 방법

PC가 불편하다면 스마트폰 앱 **손택스**에서도 동일하게 조회할 수 있습니다.

| 단계 | 내용 |
|------|------|
| 1 | 앱스토어·플레이스토어에서 **손택스** 설치 |
| 2 | 간편인증(카카오·네이버·PASS) 또는 공동인증서 로그인 |
| 3 | [조회/발급] → [근로소득 지급명세서 조회] 선택 |
| 4 | 귀속 연도 2025년 선택 후 결과 확인 |
