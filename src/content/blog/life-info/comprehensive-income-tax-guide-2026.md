매년 5월이면 돌아오는 종합소득세 신고. 직장인은 회사가 대신 처리해주지만, **프리랜서·부업자·N잡러·사업자**는 본인이 직접 신고해야 합니다. 놓치면 가산세까지 붙으니, 지금부터 미리 준비해두세요.

> 세법은 매년 변경될 수 있습니다. 정확한 내용은 [국세청 홈택스 (hometax.go.kr)](https://www.hometax.go.kr) 또는 관할 세무서에서 확인하세요.

---

## 종합소득세란?

1년간 발생한 모든 소득(근로·사업·프리랜서·이자·배당·연금·기타)을 합산해 세금을 계산하고 신고·납부하는 제도입니다. 직장인의 연말정산과 같은 개념이지만 **스스로 신고**해야 한다는 점이 다릅니다.

---

## 신고 대상 — 나는 해야 할까?

### 반드시 신고해야 하는 경우

| 해당 유형 | 구체적 사례 |
|---------|-----------|
| **프리랜서·인적용역** | 강사·작가·디자이너·개발자 등 3.3% 원천징수 후 수입 발생 |
| **부업·N잡** | 직장 다니면서 유튜브·블로그 광고수입, 배달, 쿠팡파트너스 등 |
| **사업소득자** | 개인사업자 (음식점·쇼핑몰·학원 등) |
| **임대소득** | 월세 수입 2,000만 원 초과 (2,000만 원 이하도 분리과세 선택 가능) |
| **금융소득 2,000만 원 초과** | 이자·배당 합산 2,000만 원 넘는 경우 |
| **연금소득 1,500만 원 초과** | 사적연금 합산 초과 시 |

### 직장인도 신고해야 하는 경우

- 2개 이상 직장에서 근무 (연말정산을 합산하지 않은 경우)
- 연말정산 외 부업 수입이 있는 경우
- 퇴직 후 재취업 없이 프리랜서 활동한 경우

## 신고·납부 기간

| 구분 | 기간 |
|------|------|
| **일반 신고** | 2026년 5월 1일 (금) ~ 5월 31일 (일) |
| **성실신고 확인 대상** | 2026년 6월 30일까지 |
| **분납 신청** | 납부세액 1,000만 원 초과 시 2개월 분납 가능 |

> 5월 31일이 마감일입니다. **하루라도 늦으면 무신고 가산세 20%** + 납부 지연 가산세가 추가됩니다.

## 세율 — 얼마나 내야 할까?

종합소득세는 **누진세율** 구조입니다. 소득이 높을수록 세율이 올라갑니다.

| 과세표준 (과세소득) | 세율 | 누진공제 |
|----------------|-----|--------|
| 1,400만 원 이하 | 6% | — |
| 1,400만 원 ~ 5,000만 원 | 15% | 126만 원 |
| 5,000만 원 ~ 8,800만 원 | 24% | 576만 원 |
| 8,800만 원 ~ 1억 5,000만 원 | 35% | 1,544만 원 |
| 1억 5,000만 원 ~ 3억 원 | 38% | 1,994만 원 |
| 3억 원 ~ 5억 원 | 40% | 2,594만 원 |
| 5억 원 초과 | 42% | 3,594만 원 |

> **실제 세금 = 과세표준 × 세율 − 누진공제**
>
> 예: 과세표준 3,000만 원 → 3,000만 × 15% − 126만 = **324만 원**

여기에 **지방소득세 10%**가 추가됩니다 (국세 납부 후 별도 신고).

## 신고 방법 — 홈택스 단계별 안내

### 준비물

- 공동인증서 또는 간편인증 (카카오·네이버·PASS)
- 소득 내역 (3.3% 원천징수영수증, 사업수입 장부)
- 지출 증빙 (경비처리 영수증·카드 내역)

### 홈택스 신고 절차

1. [홈택스 (hometax.go.kr)](https://www.hometax.go.kr) 접속 → 로그인
2. 상단 **"신고/납부"** → **"종합소득세"** 선택
3. **"모두채움 신고"** 또는 **"일반 신고"** 선택
   - 모두채움: 단순 프리랜서·소규모 사업자 — 국세청이 자동으로 채워준 내용 확인 후 제출
   - 일반 신고: 여러 소득 원천, 경비 직접 입력 필요한 경우
4. 소득 종류 선택 (근로·사업·기타 등)
5. 수입 금액 확인 및 경비 입력
6. 각종 공제 항목 입력 (인적공제, 연금보험료, 의료비 등)
7. 세액 계산 확인 후 **신고서 제출**
8. 납부세액이 있으면 **"납부하기"** → 계좌이체 또는 카드 납부

## 프리랜서 필수 — 경비처리로 세금 줄이기

3.3% 원천징수된 프리랜서는 **필요경비**를 인정받을수록 세금이 줄어듭니다.

### 인정되는 주요 경비 항목

| 항목 | 내용 |
|------|------|
| 장비·도구 구입비 | 카메라, 노트북, 마이크, 소프트웨어 |
| 통신비 | 업무용 휴대폰 요금 (비율 안분) |
| 교통비 | 업무 관련 이동 (영수증 필수) |
| 교육비 | 업무 관련 강의·강좌 수강료 |
| 도서비 | 업무 관련 책 구입 |
| 임차료 | 스튜디오·작업실 월세 |
| 회의비 | 거래처 식사·미팅 비용 |

> 경비로 처리하려면 **세금계산서·카드영수증·현금영수증** 등 증빙이 필수입니다.

### 단순경비율 vs 기준경비율

수입이 크지 않은 경우, 증빙 없이도 **단순경비율**로 자동 계산됩니다.

| 업종 | 단순경비율 기준 수입 | 단순경비율 |
|------|-----------------|----------|
| 프리랜서 (인적용역) | 2,400만 원 미만 | 60~70% |
| 도소매업 | 6,000만 원 미만 | 90% 내외 |
| 서비스업 | 3,600만 원 미만 | 60~70% |

수입 금액이 기준 이상이면 기준경비율(또는 장부 기재)로 신고해야 합니다.

## 절세 팁

### 1. 노란우산공제 가입

소기업·소상공인이 가입할 수 있는 공제로, **납입액 최대 500만 원까지 소득공제** 가능합니다. 절세 효과가 큰 편입니다.

### 2. 개인형 IRP 납입

연간 **900만 원 한도**로 세액공제 16.5% 적용. 실질적으로 최대 148만 원 환급 가능합니다.

### 3. 신용카드 vs 현금영수증 증빙 철저히

경비로 사용한 금액은 반드시 사업용 카드 또는 현금영수증으로 수취해두세요. 증빙 없으면 경비 인정 불가입니다.

### 4. 부가가치세 신고와 연계

일반과세자는 부가세 신고(1월·7월)에서 입력한 매입세액이 종합소득세 경비와 연동됩니다. 두 신고를 일관되게 관리해야 합니다.

## 가산세 — 놓쳤을 때 비용

| 유형 | 가산세율 |
|------|--------|
| 무신고 (일반) | 납부세액의 20% |
| 무신고 (부정) | 납부세액의 40% |
| 과소신고 | 과소납부액의 10% |
| 납부 지연 | 1일당 0.022% (연 약 8%) |

신고를 못했더라도 **기한 후 신고**를 하면 가산세가 일부 감면됩니다. 1개월 내 신고 시 50% 감면, 6개월 내 30% 감면.

## 핵심 요약

| 항목 | 내용 |
|------|------|
| 신고 기간 | **5월 1일 ~ 5월 31일** |
| 신고 방법 | 홈택스 온라인 (모두채움·일반 선택) |
| 대상 | 프리랜서·부업자·개인사업자·임대소득자 등 |

5월 1일부터 홈택스에서 신고할 수 있습니다. 4월 중에 수입·지출 내역을 미리 정리해두면 신고 당일 훨씬 수월합니다.
