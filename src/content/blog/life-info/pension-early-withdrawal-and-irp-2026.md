## IRP란 무엇인가 — 기본 개념 정리

IRP(Individual Retirement Pension, 개인형 퇴직연금)는 근로자가 퇴직할 때 받는 퇴직금을 적립하거나, 재직 중 스스로 추가 납입하여 노후 자금을 마련하는 계좌입니다.

| 구분 | 내용 |
|------|------|
| 가입 대상 | 근로자, 자영업자, 퇴직자 누구나 |
| 연간 납입 한도 | **1,800만 원** (연금저축 포함 합산) |
| 세액공제 한도 | 연금저축 포함 **최대 900만 원** |
| 세액공제율 | 총급여 5,500만 원 이하: **16.5%** / 초과: **13.2%** |
| 수령 방법 | 만 55세 이후 연금 수령 시 저율 과세 |

IRP의 핵심 장점은 납입액에 대한 세액공제와 운용 수익에 대한 과세 이연입니다. 그런데 이 혜택은 만 55세 이후 연금으로 수령할 때를 전제로 합니다. 중도에 해지하거나 인출하면 혜택이 상당 부분 사라집니다.

## 중도인출 vs 해지 — 무엇이 다른가

많은 사람이 혼동하는 부분입니다. 두 가지는 전혀 다른 개념입니다.

| 구분 | 중도인출 | 계좌 해지 |
|------|----------|-----------|
| 정의 | 특정 요건 충족 시 일부 금액 인출 | 계좌 전체를 닫고 전액 수령 |
| 계좌 유지 | 유지됨 | 폐쇄됨 |
| 적용 대상 자금 | 근로자 적립금(퇴직급여 포함) | 전 적립금 |
| 세금 | 기타소득세 **16.5%** 분리과세 | 기타소득세 **16.5%** 분리과세 |
| 세액공제 추징 | 공제받은 금액에 한해 추징 | 전액 추징 |

중도인출은 요건이 법으로 엄격히 제한되어 있고, 해지는 이유 불문 가능하지만 세금 부담이 큽니다.

## 중도인출이 허용되는 조건

근로자퇴직급여보장법 시행령에 따라 아래 사유에 해당해야만 IRP 중도인출이 가능합니다. 단순히 "돈이 필요하다"는 이유로는 인출할 수 없습니다.

| 허용 사유 | 세부 요건 |
|-----------|-----------|
| 무주택자 주택 구입 | 본인 명의 무주택 확인 필요 |
| 주거 목적 전세·보증금 | 무주택자에 한함 |
| 6개월 이상 요양 | 본인·배우자·부양가족의 의료비 |
| 파산 선고 | 법원 파산 결정 후 5년 이내 |
| 개인회생 결정 | 법원 결정 후 |
| 천재지변 피해 | 공식 재해 인정 필요 |
| 고용보험 실직급여 수급 종료 | 수급 종료 후 일정 기간 내 |

**주택 구입·임차 목적 인출은 가장 많이 활용하는 사유**입니다. 단, 무주택 세대주 요건이 핵심이므로 배우자 명의 주택이 있으면 해당되지 않습니다.

## 중도인출·해지 시 세금 얼마나 내나

### 세액공제를 받은 납입금과 운용 수익

세액공제를 받은 납입금과 그 운용 수익에 대해서는 **기타소득세 16.5%**(지방소득세 포함)가 부과됩니다.

| 재원 구분 | 세율 |
|-----------|------|
| 세액공제 받은 납입금 | **기타소득세 16.5%** |
| 운용 수익 | **기타소득세 16.5%** |
| 세액공제 받지 않은 납입금 | 비과세 (원금 해당분) |
| 퇴직급여 (IRP 이전분) | **퇴직소득세** 적용 |

### 퇴직급여 포함된 경우 별도 주의

퇴직 시 퇴직금을 IRP로 이전한 경우, 해지하면 퇴직소득세가 발생합니다. IRP에서 연금으로 수령하면 퇴직소득세의 **30~40% 감면** 혜택이 있는데, 해지하면 이 혜택이 사라집니다.

| 수령 방법 | 퇴직소득세 |
|-----------|-----------|
| 연금 수령 (10년 이하) | 퇴직소득세 × **70%** (30% 감면) |
| 연금 수령 (10년 초과) | 퇴직소득세 × **60%** (40% 감면) |
| 일시금 해지 | 퇴직소득세 **100%** |

퇴직금이 **1억 원**이고 퇴직소득세가 **500만 원**이라면, IRP에서 10년 넘게 연금으로 받으면 실제 세금은 **300만 원**으로 줄어듭니다. 그냥 해지하면 **500만 원** 전액 납부입니다.

## 실제 손해 시뮬레이션

매년 IRP에 **700만 원씩 3년간 납입**, 세액공제율 16.5% 적용한 경우를 가정합니다.

| 항목 | 금액 |
|------|------|
| 총 납입액 | **2,100만 원** |
| 3년간 받은 세액공제 합계 | **약 346만 원** (700만 원 × 16.5% × 3) |
| 가정 운용 수익 (연 4%) | **약 130만 원** |
| 해지 시 과세 대상 | 세액공제분 + 운용 수익 = **약 476만 원** |
| 기타소득세 (16.5%) | **약 79만 원** |
| 순 손실 (공제 환수 + 세금) | **약 346만 원 + 79만 원 = 425만 원** |

3년 동안 받은 세액공제 혜택이 해지 한 번으로 전부 사라지고, 추가로 운용 수익에 대한 세금까지 납부하게 됩니다.

## 손해를 줄이는 방법 — 해지 전 반드시 확인

### 1. 담보대출 활용

일부 금융기관에서는 IRP 계좌 잔액의 일정 비율을 담보로 대출해 줍니다. 계좌를 유지하면서 자금을 융통할 수 있어, 해지보다 유리한 경우가 많습니다. 금리 수준과 계좌 운용 수익을 비교해 보세요.

### 2. 납입 중단 (납입 유예)

당장 여유 자금이 없다면 추가 납입을 중단하되 계좌는 유지하는 방법도 있습니다. 기존 적립금은 그대로 운용되고, 세액공제 추징도 없습니다.

### 3. 세액공제 받지 않은 납입금부터 인출

세액공제를 신청하지 않고 납입한 금액은 인출해도 과세되지 않습니다. 연말정산에서 IRP 공제를 일부러 받지 않은 금액이 있다면, 그 부분을 먼저 빼내는 전략을 쓸 수 있습니다.

### 4. 연금저축 계좌 분리 검토

IRP와 연금저축(펀드·보험)은 별도 계좌입니다. 연금저축은 IRP보다 중도인출 요건이 상대적으로 유연합니다. 급전이 필요하다면 연금저축 계좌에서 먼저 인출을 고려해 보세요. 단, 연금저축도 세액공제분에 대해 동일하게 16.5% 기타소득세가 발생합니다.

## IRP 해지 절차

요건에 해당하거나 해지를 결정했다면 아래 절차를 따릅니다.

| 단계 | 내용 |
|------|------|
| 1단계 | IRP 가입 금융기관(은행·증권사·보험사) 방문 또는 앱 접속 |
| 2단계 | 중도인출 또는 해지 신청서 작성 |
| 3단계 | 사유 증빙 서류 제출 (중도인출의 경우) |
| 4단계 | 세금 원천징수 후 잔액 지급 |
| 5단계 | 홈택스에서 기타소득 신고 여부 확인 (분리과세이므로 별도 신고 불필요) |

중도인출 사유 증빙 서류는 사유마다 다릅니다. 주택 구입이라면 매매계약서, 요양이라면 의사 진단서 및 진료비 영수증 등을 준비해야 합니다.

## 2026년 달라진 점 — 연금 과세 체계 개편

2026년부터 연금 수령 시 과세 방식이 일부 조정되었습니다.

| 구분 | 2025년 이전 | 2026년 |
|------|-------------|--------|
| 연금소득 분리과세 한도 | 연 **1,500만 원** | 연 **1,500만 원** (유지) |
| 1,500만 원 초과 시 | 종합과세 또는 16.5% 선택 | 종합과세 또는 16.5% 선택 (유지) |
| 연금소득세율 (70세 미만) | **5.5%** | **5.5%** |
| 연금소득세율 (70~80세) | **4.4%** | **4.4%** |
| 연금소득세율 (80세 이상) | **3.3%** | **3.3%** |

현재 연금소득 과세 구조는 유지되고 있으며, 정부는 연금 과세 체계 전반에 대한 개편 논의를 진행 중입니다. 2026년 하반기 세법 개정안에 따라 변경될 수 있으므로 지속적인 확인이 필요합니다.
