## 육아휴직급여란?

육아휴직급여는 만 8세 이하 또는 초등학교 2학년 이하의 자녀를 돌보기 위해 육아휴직을 사용하는 근로자에게 고용보험에서 지급하는 급여입니다. 회사에서 주는 돈이 아니라 **국가(고용보험)에서 직접 지급**하는 지원금이므로, 회사 규모와 무관하게 신청할 수 있습니다.

### 지원 대상 요건

| 요건 | 내용 |
|------|------|
| 피보험자 기간 | 육아휴직 시작일 이전 고용보험 가입 기간 합산 **180일 이상** |
| 자녀 나이 | 만 **8세 이하** 또는 초등학교 **2학년 이하** |
| 휴직 기간 | 자녀 1명당 최대 **1년** (부모 각각 적용) |
| 고용보험 가입 여부 | 반드시 **고용보험 가입** 사업장 근로자 |

자영업자나 고용보험 미가입자는 대상에서 제외됩니다. 다만, 임의가입한 경우에는 수급 가능 여부를 고용보험 홈페이지에서 별도 확인해야 합니다.

## 일반 육아휴직급여 계산 방법

육아휴직급여는 통상임금의 **80%**를 지급하되, 상한액과 하한액이 있습니다.

| 기간 | 급여율 | 상한액 | 하한액 |
|------|--------|--------|--------|
| 1~3개월 | 통상임금 80% | 월 **250만 원** | 월 **70만 원** |
| 4개월~종료 | 통상임금 80% | 월 **200만 원** | 월 **70만 원** |

**계산 예시**
- 통상임금 월 300만 원인 근로자의 경우
  - 1~3개월: 300만 원 × 80% = 240만 원 (상한액 이하이므로 240만 원 지급)
  - 4개월~: 300만 원 × 80% = 240만 원이지만 상한 200만 원 적용 → **200만 원 지급**
- 통상임금 월 150만 원인 근로자의 경우
  - 1~3개월: 150만 원 × 80% = 120만 원 지급
  - 하한액 70만 원 이상이므로 그대로 적용

## 6+6 부모육아휴직 제도 — 둘이 쓰면 더 받는다

2024년부터 시행된 **6+6 부모육아휴직제**는 2026년에도 유지·확대됩니다. 부모 모두 육아휴직을 사용할 경우, 처음 6개월 동안 급여를 대폭 올려주는 제도입니다.

### 6+6 급여 상한액 (2026년)

| 육아휴직 개시 월 | 부모 각각 상한액 |
|----------------|----------------|
| 1개월 | 월 **250만 원** |
| 2개월 | 월 **300만 원** |
| 3개월 | 월 **350만 원** |
| 4개월 | 월 **400만 원** |
| 5개월 | 월 **450만 원** |
| 6개월 | 월 **500만 원** |

- 부모 각각 최대 6개월까지 적용
- **급여율은 통상임금의 100%** (일반 육아휴직의 80%보다 높음)
- 부부가 동시에 또는 순차적으로 사용 가능

**적용 조건**
- 자녀 생후 **18개월 이내**에 부모가 모두 육아휴직 사용
- 두 번째로 육아휴직을 시작하는 부모 기준으로 적용 (첫 번째 부모는 일반 급여 적용 → 사후 정산)

## 육아휴직급여 신청 방법

### 신청 시기

육아휴직을 시작한 날 이후, 매월 단위로 신청합니다. 신청 기한은 육아휴직이 끝난 날로부터 **12개월 이내**이며, 기한을 넘기면 급여를 받을 수 없으니 주의해야 합니다.

### 신청 절차

| 단계 | 내용 |
|------|------|
| 1단계 | 회사(사업주)에 육아휴직 신청 → 확인서 발급 |
| 2단계 | [고용보험 홈페이지](https://www.ei.go.kr) 또는 고용센터 방문 신청 |
| 3단계 | 육아휴직 확인서 + 신청서 제출 |
| 4단계 | 매월 지급 (전월분을 다음 달에 지급) |

### 온라인 신청 경로

고용보험 홈페이지 → 개인서비스 → 육아휴직급여 → 신청서 작성

공인인증서(공동인증서) 또는 간편인증(카카오·네이버 등)으로 로그인 후 신청 가능합니다.

### 제출 서류

| 서류 | 비고 |
|------|------|
| 육아휴직급여 신청서 | 온라인 작성 가능 |
| 육아휴직 확인서 | 사업주가 발급 (온라인 전송 가능) |
| 통상임금 확인 자료 | 임금대장, 급여명세서 등 |
| 출생증명서 또는 가족관계증명서 | 최초 신청 시 1회 제출 |

## 출산전후휴가 vs 육아휴직 — 차이 정리

헷갈리는 두 제도의 차이를 간단히 정리합니다.

| 항목 | 출산전후휴가급여 | 육아휴직급여 |
|------|----------------|-------------|
| 대상 | 출산 전후 여성 근로자 | 남녀 근로자 모두 |
| 기간 | 총 **90일** (다태아 120일) | 자녀 1명당 최대 **1년** |
| 급여 | 통상임금 100% (상한 월 210만 원) | 통상임금 80% (상한 월 200~250만 원) |
| 신청 주체 | 본인 | 본인 |
| 재원 | 고용보험 | 고용보험 |

출산전후휴가 이후 곧바로 육아휴직을 이어서 사용하는 것이 가능하며, 이 경우 별도로 육아휴직급여를 추가 신청해야 합니다.

## 육아기 근로시간 단축 급여 — 육아휴직 대신 쓰는 방법

육아휴직 대신 근로시간을 줄이는 **육아기 근로시간 단축** 제도도 있습니다. 2026년부터 급여 지원이 강화되었습니다.

| 항목 | 내용 |
|------|------|
| 대상 자녀 나이 | 만 **12세 이하** (초등학교 6학년 이하) — 2026년 확대 |
| 단축 가능 시간 | 주 15~35시간 (주 5시간 이상 단축) |
| 급여율 | 단축된 시간의 통상임금 **100%** (첫 5시간 기준) |
| 상한액 | 매주 단축한 5시간분: 월 **통상임금 100%** 적용 |
| 사용 기간 | 육아휴직 미사용 기간 포함 최대 **2년** |

육아휴직을 쓰기 어려운 직군이나 맞벌이 가정에서 유용하게 활용할 수 있는 제도입니다.

## 자주 묻는 질문

**Q. 계약직·파견직도 육아휴직급여를 받을 수 있나요?**
고용보험에 가입되어 있고 피보험 기간이 180일 이상이라면 계약직·파견직도 신청 가능합니다. 단, 육아휴직 종료 전에 계약이 만료되면 급여도 종료됩니다.

**Q. 남편도 육아휴직급여를 받을 수 있나요?**
남성 근로자도 동일한 조건으로 신청 가능합니다. 특히 6+6 부모육아휴직제는 아버지도 함께 사용해야 혜택이 극대화됩니다.

**Q. 쌍둥이를 낳으면 두 배로 받을 수 있나요?**
자녀별로 각각 최대 1년씩 육아휴직을 사용할 수 있습니다. 쌍둥이라면 자녀 2명 각각에 대해 1년씩 총 2년 사용이 가능하며, 급여도 별도로 지급됩니다.

**Q. 프리랜서·개인사업자도 받을 수 있나요?**
고용보험 임의가입자가 아닌 일반 자영업자·프리랜서는 육아휴직급여 대상이 아닙니다. 대신 부모급여, 아동수당 등 다른 지원제도를 활용하시기 바랍니다.
